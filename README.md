# ZKCategories
开发中常用的工具类
* 新增了KVO 防闪退,
```ruby
 pod 'ZKCategories', :subspecs => ['ZKKVOSAFE']
 ```

## 如何安装
__Requirements__
* iOS 7.0+
* macOS 10.7+

__Podfile__
```ruby
pod 'ZKCategories', :git => 'https://github.com/kaiser143/ZKCategories.git', :tag => '0.2.x'

pod 'ZKCategories', :git => 'https://github.com/kaiser143/ZKCategories.git', :commit => 'xxxx'
```
